This program draws various sierpinski shapes and other interesting 
geometric shapes.
------------------------------------------------------------------------

PROJECT TITLE: ShapesAndThings
PURPOSE OF PROJECT: Use JPanel to draw recursive geoemtric shapes and patterns
HOW TO START THIS PROJECT: Download BlueJ and open package.bluej in it. Right-click on the Runner class and click on "void main(String[] args)", then click "OK".
AUTHORS: Andrew Wang
USER INSTRUCTIONS: To change the "depth" of the pattern or the shape itself, go into the Background class and manually edit the "num" and "shape" attributes respectively.
